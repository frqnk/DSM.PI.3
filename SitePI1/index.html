Como doar:

1º Etapa

Você encaminha suas doações aos nossos pontos de coleta responsáveis, realizando também um cadastro em nosso site, podendo assim realizar um login posteriormente.

2º Etapa

É realizada a triagem das doações, separando e catalogando todos os produtos para depois enviar para o destino selecionado.

3º Etapa

Após ser encaminhado para as transportadoras, um ID de rastreio será disponibilizado, facilitando assim o acesso.
